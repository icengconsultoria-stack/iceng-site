# ICENG Consultoria — Projeto pronto para Vercel

## Requisitos
- Node.js 18 ou superior
- Conta no GitHub
- Conta na Vercel

## Rodar localmente
```bash
npm install
npm run dev
```

## Gerar build
```bash
npm install
npm run build
```

## Publicar na Vercel
1. Crie um repositório no GitHub.
2. Envie todos os arquivos deste projeto para o repositório.
3. Na Vercel, clique em **Add New > Project**.
4. Importe o repositório do GitHub.
5. Clique em **Deploy**.
6. Depois do deploy, vá em **Settings > Domains**.
7. Adicione `icengconsultoria.com.br` e `www.icengconsultoria.com.br`.
8. Copie os registros DNS que a Vercel mostrar e configure no Registro.br.

## Observações
- Os links da Hotmart já estão configurados.
- O WhatsApp e o Instagram já estão vinculados.
- Para futuras alterações, edite `src/App.jsx`.
