# Amostras de Produtos

Esta pasta contém as amostras em PDF dos produtos digitais da ICENG.

## Como adicionar amostras

1. Coloque os arquivos PDF nesta pasta com os seguintes nomes:
   - `procedimentos-amostra.pdf` - Amostra de Procedimentos Operacionais
   - `instrucoes-amostra.pdf` - Amostra de Instruções de Trabalho
   - `sst-amostra.pdf` - Amostra de Kit SST
   - `its-manual-amostra.pdf` - Amostra de ITs + Manual
   - `sgq-completo-amostra.pdf` - Amostra de SGQ Completo
   - `sgq-premium-amostra.pdf` - Amostra de SGQ Premium

2. Faça commit e push para o GitHub

3. As amostras aparecerão automaticamente no site com o botão "Ver Amostra Grátis"

## Benefícios

- ✅ Sem custos adicionais (hospedagem via GitHub)
- ✅ Rápido e econômico
- ✅ Aumenta conversão (clientes veem amostra antes de comprar)
- ✅ Reduz dúvidas sobre qualidade do produto
